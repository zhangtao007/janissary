靓号配置文件中:
1.start代表启动手动派发游客类型好友的靓号机制,1代表启动，0代表不启动
2.游客类型的靓号配置文件需要遵循以下规则:
3.首位（千万位ten_million）：1-6段注册，7-9段游客

配置文件中关键词:
ten_million //千万位,不能为空
one_million //百万位,不能为空
one_hundred_thousand //十万位,不能为空
ten_thousand //万位,不能为空
one_thousand //千位,不能为空
one_hundred //百位,不能为空
ten //十位,不能为空
one //个位,不能为空
start //是否启动,不能为空
